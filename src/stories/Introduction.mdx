# Bem-vindo ao Mri UI

Este é o sistema de design oficial para os resources da **Mri**. Ele fornece uma coleção de componentes de UI reutilizáveis, estilo consistente e diretrizes para garantir uma experiência de usuário unificada em toda a aplicação.

## Princípios Básicos

-   **Consistência:** Todos os componentes compartilham a mesma linguagem de design (cores, tipografia, espaçamento).
-   **Dark Mode First:** Projetado principalmente para interfaces escuras com acentos neon de alto contraste.
-   **Acessibilidade:** Construído pensando na acessibilidade usando primitivos do Radix UI.

## Começando

Explore os componentes na barra lateral para entender como usá-los em suas funcionalidades. Cada componente inclui controles interativos e documentação.
